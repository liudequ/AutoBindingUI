UI绑定流程：

-----------------------程序端------------------------------

1.新建文本文档，文档名称为具体Win的名称 如： Home.txt
2.文档中填写具体需要绑定的UI名称及注释，需要注意的是UI名称以项目中UIInjectionType枚举中具体类型的最后一个单词结尾。表名称没要求。
3.具体内容格式如下：

AAASprite		        -- aaa  
BBBLable 		        -- bbb
TableCCC 			    -- tableccc
{
	DDDObject			-- ddd
	EEEMessage			-- eee
	Item 				-- item			
	{
		FFFSlider 		-- fff
	}
}


-----------------------end---------------------------------


-----------------------UI端--------------------------------

1.根据程序端写的文档，修改对应UI名称。
2.打开绑定窗口：
3.拖入程序写的文档 => Read => 拖入具体UI预制件根结点 => Binding
4.等待绑定完成后，预制件 点击 Applay。

-----------------------end---------------------------------
